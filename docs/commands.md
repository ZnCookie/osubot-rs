# 命令参考

## 查询

| 命令 | 说明 |
|------|------|
| `~` | 自己（默认模式） |
| `~<模式>` / `~,<模式>` | 自己指定模式，`0`=osu! `1`=taiko `2`=catch `3`=mania |
| `where <用户名>[,模式]` | 查指定用户 |
| `where @<QQ号>[,模式]` | 查该 QQ 绑定的用户 |
| `查@<QQ用户>[,模式]` | 通过 @提及查用户 |
| `where qq=<QQ号>[,模式]` | 通过 QQ 号查绑定用户 |

## 绑定

- `绑定 <osu用户名>` — 绑定账号（IRC 鉴权开启时需在 IRC 私聊 Bot 发送验证码）
- `解绑` — 解绑（需二次确认）

## 其他

- `!help` — 显示帮助
- `!mode` — 查询当前默认模式
- `!mode <模式>` — 设置默认查询模式（`0`=osu! `1`=taiko `2`=catch `3`=mania）
- `今日高光[,模式]` — 当日高光
- `!profile [<用户名>|@<QQ号>|qq=<QQ号>]` — 个人主页卡片

## 成绩查询

```
!p|!r|!ps|!rs|!s|!ss [:<模式>] [<谱面ID>|<成绩ID>] [<用户>|@<QQ号>|qq=<QQ号>] [+<MODS>[,<条件>]] [#<N>|#<N-M>]
```

- `!p` / `!r` — 最近通过/游玩单条成绩卡片
- `!ps` / `!rs` — 最近 20 条列表
- `!s` — 谱面最佳成绩，`!ss` — 谱面所有成绩
- `!s <成绩ID>` — 按 ID 查单条成绩（输入数字 ≥ 10,000,000）
- `!p <谱面ID>` — 指定谱面的最近通过成绩
- 省略谱面 ID 时使用群内最近查询的谱面（24h 缓存）
- 用户参数支持 `@<QQ号>`、`qq=<QQ号>`、`<osu用户名>` 或群 @提及

**位置简写：** 数字 ≤200 自动识别为位置，无需 `#` 前缀
- `!p5` / `!p 5` — 等同 `!p #5`，第 5 条通过成绩
- `!ps 3-10` — 等同 `!ps #3-10`，第 3 到第 10 条
- `!p 12345` — 数字 >200 识别为谱面 ID

**条件过滤：** `key=value`（`miss` `combo` `pp` `score` `acc` `mod`）
- 操作符：`=` `==` `!=` `>` `<` `>=` `<=`
- `+MODS` 等价于 `mod=MODS`，可混用：`+HDHR,miss=1`
- 浮点键（`pp`/`acc`）`==`/`!=` 容差 0.5
- `mod=` 子集包含，`mod==` 精确集合，`mod!=` 子集取反

### 最佳成绩

```
!b|!bs [:<模式>] [<用户>|@<QQ号>|qq=<QQ号>] [+<MODS>[,<条件>]] [#<N>|#<N-M>]
```

- `!b` — 单张最佳成绩卡片（PP 最高）
- `!bs` — 最佳成绩列表（默认 20 条）
- `!b #3` — 第 3 高的成绩
- `!bs #5-10` — 第 5 到第 10 名

**位置简写：** 同成绩查询，`!b5` = `!b #5`，`!bs 2-10` = `!bs #2-10`

**条件过滤：** 同成绩查询（`miss` `combo` `pp` `score` `acc` `mod`）

### 今日 BP 查询

```
!t [:<模式>] [<用户>|@<QQ号>|qq=<QQ号>] [+<MODS>[,<条件>]] [#<N>|#<N-M>]
```

- `!t` — 今日 BP 列表（最近 24 小时内打入最佳榜的成绩）
- `!t <用户名>` / `!t @<QQ号>` — 查指定用户
- `!t :1` — taiko 模式
- `!t #3` — 单条成绩卡片
- `!t #1-5` — 第 1 到第 5 名
- `!t +HD,miss=0` — 带 mod/条件过滤

**位置简写：** 同成绩查询，`!t5` = `!t #5`，`!t 2-10` = `!t #2-10`

**条件过滤：** 同成绩查询（`miss` `combo` `pp` `score` `acc` `mod`）

### 谱面预览

- `!rv [:<模式>] [<谱面ID>|<成绩ID>|<用户>|@<QQ号>|qq=<QQ号>|#<N>] [+<MODS>[,<条件>]] [<时间> [<时间>]] [--gif|-g]` — 渲染谱面预览图
  - 参数语法同上（不含区间 `#N-M`）
  - 模式行为：显式指定 `:模式` 时使用指定模式；未指定时使用谱面原始模式（不回退用户默认模式）
  - 不带 `--gif` 时：osu! (mode 0) 输出 GIF，其余模式输出 PNG
  - 带 `--gif` 或 `-g` 时：所有模式均输出 GIF
  - 时间参数格式 `mm:ss:mmm`，支持 1 个（锚点，60s 窗口）或 2 个（区间，等距采样 4 段）
  - 示例：`!rv 01:30:000` `!rv :1 01:00:000 02:00:000` `!rv 12345 +HD 01:30:000` `!rv ZnCookie` `!rv #3`
- `!a [:<模式>] [<谱面ID>|<成绩ID>|#<N>] [<用户>|@<QQ号>|qq=<QQ号>] [+<MODS>[,<条件>]]` — 发送谱面预览音频（30s MP3）
  - 参数语法同上（不含区间 `#N-M`）
  - 模式行为：与成绩查询命令一致，显式指定 `:模式` 时使用指定模式；未指定时回退用户默认模式
  - 无参数时优先使用群内最近查询的谱面（24h 缓存）；无缓存时取自己最近一条游玩
  - 带位置/用户名/模式/过滤条件时不查缓存
  - 音频来源 `https://b.ppy.sh/preview/{beatmapset_id}.mp3`
