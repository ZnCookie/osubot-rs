# 命令参考

## 查询

| 命令 | 说明 |
|------|------|
| `~` | 自己（默认模式） |
| `~<模式>` / `~,<模式>` | 自己指定模式，`0`=osu! `1`=taiko `2`=catch `3`=mania |
| `where <用户名>[,模式]` | 查指定用户 |
| `where @<QQ号>[,模式]` | 查该 QQ 绑定的用户 |
| `查@<QQ用户>[,模式]` | 通过 @提及查用户 |
| `where qq=<QQ号>[,模式]` | 通过 QQ 号查绑定用户 |

## 绑定

- `绑定 <osu用户名>` — 绑定账号（IRC 鉴权开启时需在 IRC 私聊 Bot 发送验证码）
- `解绑` — 解绑（需二次确认）

## 其他

- `!help` — 显示帮助
- `!mode` — 查询当前默认模式
- `!mode <模式>` — 设置默认查询模式（`0`=osu! `1`=taiko `2`=catch `3`=mania）
- `今日高光[,模式]` — 当日高光
- `!profile [<用户名>|@<QQ>]` — 个人主页卡片
- `!rv [:<模式>] [<谱面ID>|<成绩ID>] [+<MODS>] [<时间> [<时间>]] [--gif|-g]` — 渲染谱面预览图
  - 不带 `--gif` 时：osu! (mode 0) 输出 GIF，其余模式 (1=taiko / 2=catch / 3=mania) 输出 PNG
  - 带 `--gif` 或短形 `-g` 时：所有模式均输出 GIF
  - 不带谱面 ID 时使用群内最近查询的谱面（24h 缓存）
  - 时间参数格式 `mm:ss:mmm`，支持 1 个（锚点，60s 窗口）或 2 个（区间，等距采样 4 段）
  - 示例：`!rv 01:30:000` `!rv :1 01:00:000 02:00:000` `!rv 12345 +HD 01:30:000`
- `!a [:<模式>] [<谱面ID>|<成绩ID>] [<用户>]` — 发送谱面预览音频（30s MP3 语音）
  - 不带 ID 时使用目标用户最近游玩的谱面（默认自己，可 `@QQ` 或带用户名）
  - `!a <成绩ID>`（≥ 10,000,000）按成绩取谱面；`!a <谱面ID>` 按谱面 ID
  - 音频来源 `https://b.ppy.sh/preview/{beatmapset_id}.mp3`

## 成绩查询

```
!p|!r|!ps|!rs|!s|!ss [:<模式>] [<谱面ID>|<成绩ID>] [<用户>] [+<MODS>[,<条件>]] [#<N>|#<N-M>]
```

- `!p` / `!r` — 最近通过/游玩单条成绩卡片
- `!ps` / `!rs` — 最近 20 条列表
- `!s` — 谱面最佳成绩，`!ss` — 谱面所有成绩
- `!s <成绩ID>` — 按 ID 查单条成绩（输入数字 ≥ 10,000,000）
- `!p <谱面ID>` — 指定谱面的最近通过成绩
- 省略谱面 ID 时使用群内最近查询的谱面（24h 缓存）

**条件过滤：** `key=value`（`miss` `combo` `pp` `score` `acc` `mod`）
- 操作符：`=` `==` `!=` `>` `<` `>=` `<=`
- `+MODS` 等价于 `mod=MODS`，可混用：`+HDHR,miss=1`
- 浮点键（`pp`/`acc`）`==`/`!=` 容差 0.5
- `mod=` 子集包含，`mod==` 精确集合，`mod!=` 子集取反

## 最佳成绩查询

```
!b|!bs [:<模式>] [<用户>] [+<MODS>[,<条件>]] [#<N>|#<N-M>]
```

- `!b` — 单张最佳成绩卡片（PP 最高）
- `!bs` — 最佳成绩列表（默认 20 条）
- `!b #3` — 第 3 高的成绩
- `!bs #5-10` — 第 5 到第 10 名

**条件过滤：** 同成绩查询（`miss` `combo` `pp` `score` `acc` `mod`）
