# 命令参考

> **全角字符归一化**：Bot 会自动将全角标点转为半角（`～`→`~`、`，`→`,`、`！`→`!`），因此使用中文输入法时无需切换即可发送命令。

## 查询

| 命令 | 说明 |
|------|------|
| `~` | 自己（默认模式） |
| `~<模式>` / `~,<模式>` | 自己指定模式，`0`=osu! `1`=taiko `2`=catch `3`=mania |
| `where <osu!用户名>[,模式]` | 查指定用户 |
| `where @<QQ用户>[,模式]` | 查该 QQ 绑定的用户 |
| `查@<QQ用户>[,模式]` | 通过 @提及查用户 |
| `where qq=<QQ号>[,模式]` | 通过 QQ 号查绑定用户 |

## 绑定

- `绑定 <osu!用户名>` — 绑定账号（IRC 鉴权开启时需在 IRC 私聊 Bot 发送验证码）
- `解绑` — 解绑（需二次确认）

## 其他

- `!help` — 显示帮助
- `!mode` — 查询当前默认模式
- `!mode <模式>` — 设置默认查询模式（`0`=osu! `1`=taiko `2`=catch `3`=mania），无效值回退到查询
- `今日高光[,模式]` — 当日高光
- `!profile [<用户名>|@<QQ用户>|qq=<QQ号>]` — 个人主页卡片

## 比赛监听

监听 osu! 多人比赛（mp）的进度与结果，自动在群内推送通知。

```
!ml <比赛ID|mp链接|比赛URL>
!ml stop <比赛ID|mp链接|比赛URL>
!ml stop all
!ml list
!ml status <比赛ID|mp链接|比赛URL>
```

- `!ml 12345678` — 开始监听比赛（支持纯数字 ID、`mp12345678`、完整 URL）
- `!ml stop 12345678` — 停止监听指定比赛
- `!ml stop all` — 停止监听本群所有比赛
- `!ml list` — 查看本群正在监听的比赛列表
- `!ml status 12345678` — 查询指定比赛的当前状态

**比赛 ID 格式：** 以下三种等价
- 纯数字：`12345678`
- `mp` 前缀：`mp12345678`
- 完整 URL：`https://osu.ppy.sh/community/matches/12345678`

**注意：**
- 每群最多同时监听 3 个比赛（可在配置中调整 `match_listen.max_per_group`）
- 默认每 8 秒轮询一次，并在启动监听后立即执行首轮检查（可在 `match_listen.poll_interval_secs` 调整）
- 比赛结束后自动推送最终结果卡片并停止监听
- 不支持 lazer 房间 URL（`/multiplayer/rooms/...`），仅支持旧版 mp 链接

## 成绩查询

### 通用语法

```
<命令> [:<模式>] [<谱面ID>|<成绩ID>|<osu!用户名>|@<QQ用户>|qq=<QQ号>] [+<MODS>[,<条件>]] [#<N>|#<N-M>] [<mm:ss:mmm> [<mm:ss:mmm>]] [--gif|-g]
```

各命令的参数子集见下表：

| 命令 | 谱面ID | 成绩ID | 用户 | 位置 | 区间 | +MODS/条件 | 时间 | --gif |
|------|:------:|:------:|:----:|:----:|:----:|:----------:|:----:|:-----:|
| `!p` / `!r` / `!s` | ✓ | ✓ | ✓ | ✓ | — | ✓ | — | — |
| `!ps` / `!rs` / `!ss` | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | — | — |
| `!b` | ✓ | ✓ | ✓ | ✓ | — | ✓ | — | — |
| `!bs` | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | — | — |
| `!t` | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | — | — |
| `!rv` | ✓ | ✓ | ✓ | ✓ | — | ✓ | ✓ | ✓ |
| `!a` | ✓ | ✓ | ✓ | ✓ | — | ✓ | — | — |

### 命令说明

**最近成绩**
- `!p` / `!r` — 最近通过/游玩单条成绩卡片
- `!ps` / `!rs` — 最近通过/游玩列表（默认 20 条）

**谱面成绩**
- `!s` — 谱面最佳成绩
- `!ss` — 谱面所有成绩（默认 20 条）
- `!s <成绩ID>` — 按 ID 查单条成绩（输入数字 ≥ 10,000,000）
- `!p <谱面ID>` — 指定谱面存在于最近通过列表（`!ps`）的成绩（返回最新的一条）
- 省略谱面 ID 时使用群内最近查询的谱面（24h 缓存）

**最佳成绩**
- `!b` — 单张最佳成绩卡片（PP 最高）
- `!bs` — 最佳成绩列表（默认 20 条）
- `!b #3` — 第 3 高的成绩，`!bs #5-10` — 第 5 到第 10 名
- `!b <谱面ID>` — 在最佳 200 中筛选该谱面的成绩（返回 PP 最高的一条）
- `!bs <谱面ID>` — 同上，以列表方式显示
- `!b <成绩ID>` — 按 ID 查单条成绩（输入数字 ≥ 10,000,000）

**今日 BP**
- `!t` — 今日 BP 列表（最近 24 小时内打入最佳榜的成绩，默认最多 20 条）
- `!t #3` — 单条成绩卡片，`!t #1-5` — 第 1 到第 5 名

**谱面预览**
- `!rv` — 渲染谱面预览图
  - 不带 `--gif`：osu! (mode 0) 输出 GIF，其余输出 PNG
  - 带 `--gif` 或 `-g`：所有模式输出 GIF
  - 时间格式 `mm:ss:mmm`（分钟支持 1-2 位）：1 个（锚点，60s 窗口）或 2 个（区间，等距采样 4 段）
  - 示例：`!rv 01:30:000` `!rv :1 01:00:000 02:00:000` `!rv 12345 +HD 01:30:000`
- `!a` — 发送谱面预览音频

**模式行为**
- 显式指定 `:模式` 时使用指定模式；未指定时回退用户默认模式

**用户参数**：支持 `@<QQ号>`、`qq=<QQ号>`、`<osu用户名>` 或群 @提及

### 位置简写

数字 ≤200 自动识别为位置，无需 `#` 前缀：
- `!p5` / `!p 5` — 等同 `!p #5`，第 5 条通过成绩
- `!ps 3-10` — 等同 `!ps #3-10`，第 3 到第 10 条
- `!p 12345` — 数字 >200 识别为谱面 ID

### 条件过滤

`key=value`（`miss` `combo` `pp` `score` `acc` `mod` `ar` `od` `cs` `hp` `star` `bpm`）
- 操作符：`=` `==` `!=` `>` `<` `>=` `<=`（也支持全角：`＝` `！` `＞` `＜`）
- `+MODS` 等价于 `mod=MODS`，可混用：`+HDHR,miss=1`
- `==`/`!=` 容差因字段而异：
  - `pp`/`bpm`：0.5
  - `ar`/`od`/`cs`/`hp`/`acc`/`star`：0.01
- `mod=` 子集包含，`mod==` 精确集合，`mod!=` 子集取反
