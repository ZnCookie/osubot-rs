# 命令参考

## 查询

| 命令 | 说明 |
|------|------|
| `~` | 自己（默认模式） |
| `~<模式>` / `~,<模式>` | 自己指定模式，`0`=osu! `1`=taiko `2`=catch `3`=mania |
| `where <用户名>[,模式]` | 查指定用户 |
| `where @<QQ号>[,模式]` | 查该 QQ 绑定的用户 |
| `查@<QQ用户>[,模式]` | 通过 @提及查用户 |
| `where qq=<QQ号>[,模式]` | 通过 QQ 号查绑定用户 |

## 绑定

- `绑定 <osu用户名>` — 绑定账号（IRC 鉴权开启时需在 IRC 私聊 Bot 发送验证码）
- `解绑` — 解绑（需二次确认）

## 其他

- `!help` — 显示帮助
- `!mode` — 查询当前默认模式
- `!mode <模式>` — 设置默认查询模式（`0`=osu! `1`=taiko `2`=catch `3`=mania）
- `今日高光[,模式]` — 当日高光
- `!profile [<用户名>|@<QQ>]` — 个人主页卡片

## 成绩查询

```
(!p|!r|!ps|!rs) [:<模式>] [<用户>] [+<MODS>[,<条件>]] [#<N>|#<N-M>]
(!s|!ss) [:<模式>] [<谱面ID>] [<用户>] [+<MODS>[,<条件>]] [#<N>|#<N-M>]
```

- `!p` / `!r` — 最近通过/游玩单条成绩卡片
- `!ps` / `!rs` — 最近 20 条列表
- `!s` — 谱面最佳成绩，`!ss` — 谱面所有成绩
- `!s <成绩ID>` — 按 ID 查单条成绩（ID ≥ 10,000,000 自动识别）
- 省略谱面 ID 时使用群内最近查询的谱面（6h 缓存）

**条件过滤：** `key=value`（`miss` `combo` `pp` `score` `acc` `mod`）
- 操作符：`=` `==` `!=` `>` `<` `>=` `<=`
- `+MODS` 等价于 `mod=MODS`，可混用：`+HDHR,miss=1`
- 浮点键（`pp`/`acc`）`==`/`!=` 容差 0.5
- `mod=` 子集包含，`mod==` 精确集合，`mod!=` 子集取反

```
!p :2 ZnCookie #5
!p +HDHR                          # 等价于 !p mod=HDHR
!p +HDHR,miss=1
!ps ZnCookie miss=1,combo=500
!ps acc>=99
!ps mod==DT
!ps mod==NM                       # 精确匹配无 mod
!ps mod!=DT
!r :3 miss=1 5                    # 5 等价于 #5
!s :1 123456 ZnCookie +DT,miss=1
!ss 123456 #2-10
```
