## 报告漏洞

如果您发现安全漏洞，请通过以下方式报告：

1. **不要** 在公开的 GitHub Issues 中报告安全漏洞
2. 请发送邮件至：`zncookie@outlook.com`
3. 或使用 [GitHub Security Advisories](https://github.com/zncookie/osubot-rs/security/advisories/new)

请包含以下信息：
- 漏洞描述
- 重现步骤
- 影响范围
- 建议的修复方案（如果有）

## 安全响应时间

- **确认收到**：48 小时内
- **初步评估**：7 天内
- **修复发布**：根据严重程度，通常在 30 天内
